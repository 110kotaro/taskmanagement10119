# カタログにあって現在のアプリにない機能

## 1. タスクの登録・編集

### 1.1 カスタム種別の入力
- **カタログ**: 「その他」を選択した場合、カスタム種別を入力できる
- **現状**: `customTaskType`フィールドは存在するが、UIから削除済み（「その他」のみ選択可能）
- **不足**: カスタム種別の入力フィールド

### 1.2 カスタムラベル（優先度）
- **カタログ**: ラベルは「重要、普通、低め、なし」の4段階から設定し、カスタムもできる
- **現状**: `customPriority`フィールドは存在するが、UIから削除済み
- **不足**: カスタム優先度の入力フィールド

## 2. タスクの表示

### 2.1 「未着手、進行中、期限切れが一目で分かるページ」
- **カタログ**: 別で、未着手、進行中、期限切れが一目で分かるページがある
- **現状**: タスク一覧ページでフィルター機能はあるが、専用ページはない
- **不足**: 期限切れタスク専用のページ（視覚的に分かりやすい表示）

## 3. タスクの管理

### 3.1 管理者権限を持つものは全タスクの通知を受け取れる
- **カタログ**: 管理者権限を持つものは全てのタスク追加、タスク編集の通知が受け取れる
- **現状**: 通知はプロジェクトメンバー/チームメンバーにのみ送信される
- **不足**: システム管理者（`UserRole.Admin`）が全タスクの通知を受け取る機能

## 4. リマインダー機能

### 4.1 メール通知
- **カタログ**: 設定時間にアプリを開いていなかった場合、メールにて通知
- **現状**: WebPush通知のみ実装されている
- **不足**: メール通知機能（Firebase Cloud Functionsでメール送信）

## 5. 通知・お知らせ設定

### 5.1 管理者の全タスク通知
- **カタログ**: 管理者権限を持つものは全てのタスク追加、タスク編集の通知が受け取れる
- **現状**: 通知設定に管理者向けの全タスク通知設定がない
- **不足**: 管理者向けの全タスク通知設定と実装

## 6. チーム管理

### 6.1 共同管理者の任命
- **カタログ**: 管理者は、共同管理者を任命でき、共同管理者の権限は管理者と同等
- **現状**: チームの役割は`owner`（オーナー）と`admin`（管理者）のみ。`admin`は既に管理者権限を持つが、「共同管理者」という概念はない
- **不足**: 共同管理者の概念は既に`admin`で実現されているが、カタログの「共同管理者を任命」という表現に合わせたUI/説明の改善が必要かもしれない

## 7. その他

### 7.1 プロジェクト単位でファイルを共有できる箱
- **カタログ**: プロジェクト単位でファイルを共有できる箱がある
- **現状**: プロジェクト詳細ページでファイルのアップロード・ダウンロード・削除機能が実装されている
- **不足**: なし（実装済み）

### 7.2 コメント検索機能の詳細
- **カタログ**: コメントタブでは検索機能があり、入力された文字が含まれているコメントを表示する。プロジェクト名を入力した場合は、そのプロジェクト下のタスクに対するコメントを表示する
- **現状**: コメント検索機能は実装されているが、プロジェクト名での検索は実装済み
- **不足**: なし（実装済み）

---

## まとめ

### 完全に不足している機能
1. **カスタム種別の入力**（「その他」選択時）
2. **カスタムラベル（優先度）の入力**
3. **期限切れタスク専用ページ**
4. **管理者の全タスク通知機能**
5. **メール通知機能**（リマインダー）

### 実装済みの機能（カタログ要件を満たしている）
1. **ポモドーロ機能の詳細動作**
   - 休憩スキップボタン（`skipBreak()`メソッド実装済み）
   - 5分経過後の通知（`onBreakComplete()`実装済み）
   - 5分ごとの再開通知（`showResumePrompt()`実装済み）
2. **ガントチャートでのドラッグ編集**
   - バー全体のドラッグ（`onBarDragStart()`実装済み）
   - 左端・右端のドラッグ（`onDragStart()`, `onDragEndStart()`実装済み）
   - 期間変更の保存機能実装済み
3. **コメント検索機能**
   - 文字列検索機能実装済み
   - プロジェクト名でのフィルタリング実装済み
4. **プロジェクト単位でのファイル共有**
   - ファイルアップロード・ダウンロード・削除機能実装済み

### 概念的に実装済みだが説明が必要な機能
1. **共同管理者の任命**（`admin`ロールで実現されている）
